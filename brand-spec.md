## Tipografía

- **Display:** Poppins, 600–700 weight, `-0.02em` tracking
- **Body:** Inter, 400–500 weight
- **Mono:** SF Mono, ui-monospace

## Layout

- Max width: 1180px
- Grid base: 8pt
- Section spacing: 96px desktop / 68px tablet / 48px phone
- Radii: 10px (sm), 16px (md), 24px (lg)

## Dirección Visual (de la referencia)

- Hero oscuro gris carbón con texto blanco y CTA
- Secciones de contenido en blanco
- Cards limpias para servicios/productos
- Jerarquía tipográfica fuerte
- Accent verde (#4CAF50) para acciones — alineado con el logo de marca
- Grises variados para fondos secundarios, footer, hero
- Sensación profesional y de confianza
